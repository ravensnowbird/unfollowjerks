== Unfollow Jerks

This is the source code of unfollowjerks i made as a hobby. Feel free to clone and have your own personal setup

Files you should change:

* config/initializers/environment_variables.rb  "YOUR TWITTER API KEY" , "YOUR TWITTER API SECRET"

* app/models/user.rb "YOUR TWITTER CONSUMER KEY" , "YOUR TWITTER API SECRET"

* config/application.rb "YOUR MailChimp EMAIL / USERNAME" , "YOUR MailChimp Password"
